### 插入数据
* insert优化
>1、批量插入
```mysql
insert into 表名 values (字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),...;
```  
> 2、手动事务提交
```mysql
start transaction ;
insert into 表名 values (字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),...;
insert into 表名 values (字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),...;
insert into 表名 values (字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),(字段值1,字段值2,字段值3,...),...;
commit ;
```
